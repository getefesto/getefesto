Build REST applications quickly and without headaches.

Developed with security, performance and ease-of-use in mind.

Efesto is a REST microserver that can deliver an API in minutes. It takes care of authentication, permissions and kickstarts you by providing a simple way to build a data structure and the means to expose it.

Written in Python on top of Falcon and backed by PostgreSQL

It uses JSON+Siren as output format

Efesto is an open source project, released under the terms of the General Public License v.3
